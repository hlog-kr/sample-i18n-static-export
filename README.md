# 사용

- `git clone https://github.com/hlog-kr/sample-i18n-static-export.git` 으로 저장소를 복제합니다.

- 프로젝트 폴더에서  `npm install` 로 필요한 패키지를 설치합니다.

- `npm run dev` 로 브라우저에서 동작 확인이 가능합니다.

- `npm run build` 로 정적 파일 생성을 확인할 수 있습니다.
